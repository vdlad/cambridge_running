## ğŸ¤ Contributing

We welcome contributions from runners of all levels! Help us build the most comprehensive collection of Cambridge running routes.

### How to Contribute

1. **Fork this repository**
2. **Create a new route file** following our naming convention: `routes_[area_name].md.html`
3. **Use our template structure** (see existing files for examples)
4. **Submit a Pull Request** with your changes

### What to Include in Route Submissions

- **Route Name & Description**: Clear, descriptive title and overview
- **Distance & Difficulty**: Accurate measurements and skill level
- **Starting Point**: Specific location with landmark references
- **Surface Type**: Paved, trail, mixed, etc.
- **Highlights**: Notable scenery, landmarks, or features
- **Safety Notes**: Any important safety considerations
- **GPS Coordinates**: If available (highly encouraged!)
- **Photos**: Scenic spots or key waypoints (optional)

### Route File Format

Each route file should be named `routes_[descriptive_name].md.html` and include:

```html
---
layout: default
title: Your Route Collection Name
permalink: /your_route_name.html
---

<!DOCTYPE html>
<html lang="en">
<!-- Your HTML content here -->
</html>
```

### Example Contributions We'd Love

- **Neighborhood-specific routes**: Porter Square, Kendall Square, etc.
- **Seasonal routes**: Fall foliage paths, winter-friendly routes
- **Training-focused routes**: Hill repeats, tempo run loops
- **Accessibility routes**: Wheelchair/stroller-friendly paths
- **Multi-distance options**: Short (1-3 mi), Medium (3-6 mi), Long (6+ mi)
