**Friday Loop on the Charles** ![BU to Mass Ave - Photo from BU, photo links](http://www.bu.edu/careers/files/2017/04/BU-Bridge-Rehabilitation-01.jpg) # Overview It is tradition to run this loop every Friday. It is a quintesential Boston run that takes you by MIT, BU and the Esplanade. Since it is around the river, everything is very flat. The main surface is asphalt shared use path. There is a decent amount of soft surface in the form of single track on the side of the path and some crushed gravel sections. The esplande gets plowed decently fast and is well illuminated so it is a solid run year round. The naming convention is driven by the bridges you use to cross the river. Forward is defined as BU first. [Home](http://web.mit.edu/sgoldy/www/routes/cambridge/routes_cambridge.md.html) # Variations Friday is traditionally 3-6 miles. This is also a great spot to get back into running. 1. BU to Mass Ave (Morning Loop) - 3 miles 2. Extended Morning Loop - 4 miles 3. BU to Longfellow - 5 miles 4.
